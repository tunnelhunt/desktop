# Tunnelhunt Desktop Client

Легковесный, кроссплатформенный клиент для управления туннелями Tunnelhunt, работающий в системном трее. Написан на базе **Tauri v2** (Rust + HTML/CSS/JS) и поддерживает запуск на **Linux, macOS и Windows**.

Интерфейс и поведение приложения полностью повторяют оригинальную macOS-версию, предоставляя удобное всплывающее меню управления пресетами прямо из панели задач.

---

## Возможности

- **Работа исключительно в трее (Tray App):** Приложение не занимает место на панели задач/Dock-панели и отображается в виде поповера под иконкой трея.
- **Сворачивание при потере фокуса:** Окно интерфейса автоматически закрывается, если кликнуть в любую другую область экрана.
- **Поддержка SSH-ключей:** Быстрый выбор кастомных приватных SSH-ключей через нативный диалог операционной системы.
- **Автоматическое переподключение:** При падении SSH-соединения клиент автоматически предпринимает до 5 попыток переподключения с интервалом в 3 секунды.
- **Уведомления**: Отображение системных пуш-уведомлений при старте и отключении туннелей.
- **Логирование ошибок**: Подробные сообщения о причинах сбоев выводятся прямо в блоке соответствующего пресета.
- **Копирование ссылок в один клик**: Рядом с активным туннелем выводится сгенерированная ссылка `https://...` с удобной кнопкой быстрого копирования.

---

## Требования

1. **SSH-клиент**: На вашей системе должен быть установлен стандартный клиент SSH (в Linux и macOS есть по умолчанию, в Windows 10/11 должен быть включен компонент OpenSSH).
2. **Поддержка системного трея (для Linux)**:
   - Приложение жестко привязано к системному трею. 
   - Если вы используете **GNOME** (например, в стандартной Ubuntu), убедитесь, что у вас установлено и включено расширение [AppIndicator and KStatusNotifierItem Support](https://github.com/ubuntu/gnome-shell-extension-appindicator).
   - При запуске в окружении без поддержки трея приложение выведет предупреждающее диалоговое окно и завершит работу.

## Установка и запуск

Готовые сборки для всех платформ автоматически собираются через GitHub Actions при публикации новых релизов:
- **Linux**: `.deb` пакеты для Debian/Ubuntu и универсальные `.AppImage`.
- **macOS**: `.dmg` образы и `.app` архивы.
- **Windows**: `.msi` установщики и портативные `.exe`.

## Локальная разработка и сборка

Для сборки приложения вам потребуются установленные **Rust** (cargo) и **Node.js** (npm).

### 1. Установка зависимостей сборки (только для Linux)
Для сборки под Linux установите библиотеки веб-движка:
```bash
sudo apt-get update
sudo apt-get install -y libgtk-3-dev libwebkit2gtk-4.1-dev libappindicator3-dev librsvg2-dev patchelf
```

### 2. Установка npm-пакетов
```bash
npm install
```

### 3. Запуск в режиме разработки
Запуск приложения в режиме live-reload:
```bash
npm run tauri dev
```

### 4. Компиляция релизной версии
Сборка готовых пакетов под вашу текущую платформу:
```bash
npm run tauri build
```
Скомпилированные файлы будут находиться по пути `src-tauri/target/release/bundle/`.
